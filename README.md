## 授课教师

由于课程内容较多，这门课会由两位老师授课。下面两位是某教学班的情况：

- 彭福军
讲授《材料力学I》第六章及之前的内容，即《简单的超静定问题》之前的内容。

- 白玉超
讲授《材料力学I》第七章及之后的内容，即《应力状态》之前的内容；并讲授《材料力学II》和《弹性力学简明教程》中教学大纲所列的内容。
